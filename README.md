# Order Organizer

## Overview
This project is a command-line Python tool that allows users to enter and organize customer orders. It collects customer names and order totals, then calculates and displays total sales.

## Features
- Collects customer order data through user input
- Validates numeric input to prevent errors
- Stores orders in a structured format
- Calculates total sales automatically
- Displays results with clean currency formatting

## How It Works
1. The user enters a customer name
2. The user inputs the order total
3. The program stores the order
4. The process repeats until the user types "done"
5. A summary of all orders and total sales is displayed

## Example Output

Enter customer name (or type done): Stephanie
Enter order total: $25

Enter customer name (or type done): Heidi
Enter order total: $40

Order Summary
Stephanie: $25.00
Heidi: $40.00

Total sales: $65.00

## Technologies Used

## Purpose
This project was built to simulate a real-world business tool for managing and summarizing customer orders. It demonstrates core Python concepts such as loops, conditionals, input validation, and data handling.
